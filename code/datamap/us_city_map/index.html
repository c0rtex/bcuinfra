Features

  • ease to use for everyone
  • small file size, low overhead (webtraffic)
  • innovative & unique Flash Map Interface
  • URL & PHP & Mysql linking
  • self selection of colors, look & feel, captions and more - all 100% customizable!
  • customization via XML; no flash modifications required
    Any setting can be achieved using an external XML file; this allows integration of a flash map into any application with easy and dynamic modification of map information
  • customization popup-windows and tooltip
  • compatible with all scripting languages like ASP, PHP, ColdFusion, Perl, etc.
    Use any programming language to create an XML source file and transfer it to your Flash object, or simply use an XML file from your server

Configuration and customization via xml

All adjustments and customizations are made in the usa.xml file. It can be opened with an standard text editor, like Windows Notepad, Notepad++ or Macromedia Dreamveawer. Certainly, you can use any other web editors which are available at the market. The configurations and customizations are made on your local computer, after configuration and customization you must upload only the edited files to your webserver directory. Thats it! 

<config>
  <background_color>ffffff</background_color>
  <button_back_text>BACK</button_back_text>
  <sound>on</sound>  
</config>

<map>
<state>
  <id>1</id>
  <name>Alabama</name>
  <URL>maps/al.swf</URL>
  <popup>text</popup>
  <color1>7798BA</color1>
  <color2>366CA3</color2>
  <xml>maps/al.xml</xml>
  <image>photo.jpg</image>  
  <url>your_page.html</url>
</state>
...
</map>




 
<config>
 <background_color>ffffff</background_color> Color of a background
 <button_back_text>BACK</button_back_text> Text on button BACK
 <sound>on</sound> Sound on/off
</config>
   
<map>
<state>
  <id>1</id> The identifier of state
  <name>Alabama</name> Name of state
  <URL>maps/al.swf</URL> For parameter URL it is possible to appoint the following:
URL = "maps/al.swf" - means the map of region opens
URL = "" - there is no data - state inactive.
  <popup>text</popup> The text for a pop-up window at rollover the mouse on state. To break line press "Enter", or use a code || or code &lt;BR&gt;
  <color1>7798BA</color1> Color of state
  <color2>366CA3</color2> State rollover color
  <xml>maps/al.xml</xml> Path to a XML-file with the information for this state
  <image>photo.jpg</image> Name of a image file. It can be the image in the size 100*100 px (jpg). This image will be displayed in popup-window.
  <url>your_page.html</url> Path to a XML-file with locations data.
</state>
... ...

  • To define coordinates of hotspot

    Addition of a point
    1. Enter the name of a new point.
    2. Click on a map in any place.

    Editing
    1. Click on a point.
    2. Edit the information.
    3. Click 'Update Map'.

    Moving
    1. Click and hold on point.
    2. Drag point.

    Removing
    1. Click on a point.
    2. Click 'Delete'.



    After modification it is necessary to copy a XML-code and to replace all code in a file XML for this region.
