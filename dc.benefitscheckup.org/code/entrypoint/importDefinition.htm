Entry Point Import CSV Column Definitions

 

Column Name

Column Definition

entrypointgroup_id

This value must be numeric and represent an existing entrypointgroup. It will exist in the entrypoint export for existing entrypoints. You will need to look up and add the corresponding entrypointgroup_id when entering new entrypoints. Currently, an entrypointgroup and entrypoint association is added when a new entrpoint is imported. When an entrypoint is reused, a check is done to ensure that an association exists between the entrypoint and the assigned entrypointgroup. (An enhancement worth considering is to allow both additions and replacements of current entrypointgroup and entrypoint associations via the import. Additional coding will be needed to enable this feature.) Required.

entrypoint_id

This value must be numeric. It will exist in the entrypoint export for existing entrypoints. Leave the cell blank to indicate that this row represents a new entrypoint. An entrypoint_id will be auto-generated if the cell is blank.

name

This cell can be alphanumeric and can contain punctuation. Required.

subname

This cell can be alphanumeric and can contain punctuation.

address1

This cell can be alphanumeric and can contain punctuation.

address2

This cell can be alphanumeric and can contain punctuation.

address3

This cell can be alphanumeric and can contain punctuation.

city

This cell can be alphanumeric and can contain punctuation.

state_id

The data in this cell must have 2 alpha characters that represent a state listed in DC-->zip-->states-->list.

zipcode

The data in this cell must have 5 digits and should not contain any punctuation.

zipcode_plus4

Any value input in this cell must have 4 digits and should not contain any punctuation. Please do not prefix the 4 digits with a dash (-).

email

This cell can be alphanumeric and can contain punctuation.

url

This cell can be alphanumeric and can contain punctuation.

hours_display_id

This cell should have a value from the entrypoint export. A blank cell here indicates that the entrypoint it is associated with is new. A corresponding hours_display_id will be auto-generated.

hours

This cell can be alphanumeric and can contain punctuation.

info_display_id

This cell should have a value from the entrypoint export. A blank cell here indicates that the entrypoint it is associated with is new. A corresponding info_display_id will be auto-generated.

info

This cell can be alphanumeric and can contain punctuation.

contact_title

This cell can be alphanumeric and can contain punctuation.

contact_first

This cell can be alphanumeric and can contain punctuation.

contact_middle

This cell can be alphanumeric and can contain punctuation.

contact_last

This cell can be alphanumeric and can contain punctuation.

contact_suffix

This cell can be alphanumeric and can contain punctuation.

privatephone

This cell can be alphanumeric and can contain punctuation.

notes

This cell can be alphanumeric and can contain punctuation.

voice

This cell can be alphanumeric and can contain punctuation.

fax

This cell can be alphanumeric and can contain punctuation.

tty

This cell can be alphanumeric and can contain punctuation.

toll_free

This cell can be alphanumeric and can contain punctuation.

translation

This cell can be alphanumeric and can contain punctuation.

spanish

This cell can be alphanumeric and can contain punctuation.

active_flag

This cell accepts a 0 or 1 where 0 is an inactive entrypoint and 1 is an active entrypoint. No punctuation or alpha characters allowed. Required.
