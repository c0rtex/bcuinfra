Mind and Body Connection

When we are living with a chronic condition, we have many symptoms. You can see some of these symptoms on this cycle.

All the things on this cycle can contribute to symptoms. Worse yet, they can feed on each other. Depression causes tiredness, stress causes tense muscles and pain, and so on. This becomes a vicious cycle that gets worse and worse unless we find a way to break the cycle. No matter where we break the cycle, we can stop things from feeding on themselves and getting worse and we can improve our discomfort. If we understand how these things on the cycle contribute to increased symptoms, we can learn techniques that help break the cycle at these various points.
