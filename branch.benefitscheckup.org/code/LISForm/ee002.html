Should You Use This Application?

Not everyone will be able to use the online Application For Help With Medicare Prescription Drug Plan Costs. You must answer a few questions to help determine if you should use this Internet form. Any time there is a link at the end of a question that says "More Info," you can follow that link to get help with that question.

The OMB control number for this application is 0960-0705; expiration date 5/31/2008.
